# Tigerbeetle

## Overview

Tigerbeetle is a high-performance distributed database designed for financial transactions and accounting.

## Used By

- **Kernel** — write for financial operations (not MVP)

## What It Stores

- Financial transactions (transfers, balances)
- Account state and ledger entries

## Why Tigerbeetle

- Purpose-built for financial workloads
- ACID transactions with high throughput
- Distributed and fault-tolerant
- Designed for correctness (double-entry bookkeeping)
